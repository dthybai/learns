vuex入门

vuex安装

      使用npm安装：npm install vuex --save
      使用yarn安装：yarn add vuex

vuex使用方法

      在src目录下创建一个store文件
      在store文件夹中新建一个index.js
      在index.js文件中引入vue和vuex（import Vue from 'vue'; and import Vuex from 'vuxe'）
      创建vuex实例( const store = new Vuex.store({}))
      导出vuex实例( export default store)
      在main.js文件中引入该文件，在文件里面添加 import store from ‘./store’;，再在vue实例全局引入store对象；

接下来就可以编写vuex业务代码

      vuex中的数据源（state） 获取数据源使用（）
      vuex 4种方法（这些方法都要放在vuex实例中） 1.state 定义初始状态 state:{count:1} 2.getters 对值进行改变（类似于计算属性computed） 3.mutations 修改store中的值唯一的方法就是提交mutation来修改 4.actions 官方不建议我们直接修改store的值，所以需要通过提交actions，然后再通过mutation修改

为了不是用this.$store.state.count这种方式获取值，vuex提供了三种方法获取

      1.在vue文件中引入 import { mapState, mapMutation, mapAction, mapGetters} from 'vuex'
      2.放在computed里执行 computed:{...mapState({ count1.state => state.count })}
